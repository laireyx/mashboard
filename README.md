# Mashboard

A desktop app starter using Node.js, pnpm, Vite, React, and Electron.

## Scripts

```bash
pnpm install
pnpm dev
pnpm build
pnpm package
```

## Project Layout

```text
src/main       Electron main process
src/preload    Safe bridge exposed to the renderer
src/renderer   React app rendered by Vite
```
